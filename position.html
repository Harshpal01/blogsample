I am the main container box , positioned relatively ,
This is default positioning or what we call static positioning
This is absolute styling
Relative positioning means the element is position relative to itself
Absolute means positioned relative to its nearest positioned parent element
When an element is positioned absolutely, it is taken out of the normal flow and the other elements fill in its gap. This allows it to have a z-index property, and means that it can overlay over other elements.
Basically, absolute elements are on a different layer. I've illustrated this by giving the absolute elements a box-shadow - like they're 'above' the other elements! Try deleting the background-color property of this element!
Fixed positioning means that I am positioned relative to the browser window so that when you scroll , I'll be right there.
